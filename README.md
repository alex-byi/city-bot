# Telegram Bot 

### Prerequisites

* Java 1.8 or newer.
* Apache Maven

### Usage
```
$ git clone https://github.com/alex-byi/city-bot.git
$ cd city-bot/
$ mvn package
$ java -jar target/telegram-1.0-SNAPSHOT.jar
```
Now you can use the bot. 
Connect to the bot and send him a message with the name of the city in Russian.
The bot will return a description.

### Maintenance

Just go to http://127.0.0.1:8080/

And you will see all the cities that were imported into the database at startup.
Database management through a Web interface is intuitive.
